# Proiect-Ingineria-Sistemelor
Acest proiect are ca scop identificarea sistemelor dinamice folosind mai multe metode clasice și moderne din domeniul Ingineriei Sistemelor. 
Sunt analizate sisteme de ordinul II, atât liniare cât și neliniare, utilizând date obținute prin simulare în MATLAB/Simulink, în prezența zgomotului și a efectelor de cuantizare.
Proiectul este structurat în patru etape principale, fiecare corespunzând unei metode diferite de identificare și validare a modelelor matematice.
